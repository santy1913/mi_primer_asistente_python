# mi_primer_asistente_python
Proyecto: Sistema de Gestión - Refugio "Patitas Felices"
📝 Descripción
Este es mi primer programa escrito en Python. Consiste en un menú interactivo que permite gestionar las operaciones básicas de un refugio de animales: ver el estado del refugio, controlar la capacidad, registrar adopciones y calcular costos de vacunación con IVA. El objetivo de este proyecto fue aprender sobre bucles, condicionales y cómo organizar código en funciones.
